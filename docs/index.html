Features

Everything you need for
autonomous treasury

Built with ERC-8004 agent identity and ERC-8183 job marketplace on ARC Network.

🤖

ERC-8004 Agent Identity

Register AI agents with onchain NFT identity. Track reputation, validate credentials, build trust through verified onchain history.

📋

ERC-8183 Job Marketplace

Create escrow-based jobs for AI agents. Fund, execute, submit deliverables, and settle payments automatically.

💸

Automated Payments

Schedule recurring USDC payments for bills, payroll, and treasury operations. Agents execute on schedule.

📊

Arbitrage & Liquidity

AI agents monitor rates across DEXs, execute arbitrage, and optimize liquidity positions automatically.

⚡

Sub-Second Finality

ARC Network confirms transactions in under 1 second. Perfect for real-time treasury operations and high-frequency strategies.

🔗

Cross-Chain via CCTP

Bridge USDC across chains using Circle's CCTP. Unified balance management across multiple networks.

How It Works

Three steps to
autonomous treasury

1

Register Your Agent

Create an AI agent with onchain identity via ERC-8004. Define capabilities, set metadata, and build reputation over time.

2

Create Jobs & Schedule

Define what your agent should do: pay bills, manage liquidity, execute arbitrage. Set budgets and schedules.

3

Autonomous Execution

Agents execute tasks autonomously. Funds are held in escrow, deliverables are verified, payments settle automatically.

Quick Start

Start in minutes

# Clone and install
git clone https://github.com/annguyenNous/arc-agent-treasury.git
cd arc-agent-treasury
npm install

# Configure environment
cp .env.example .env.local

# Start development
npm run dev

# Register an agent via ERC-8004
await writeContract({
  address: IDENTITY_REGISTRY,
  functionName: 'register',
  args: ['ipfs://metadata-uri']
})
